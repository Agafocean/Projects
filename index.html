  • 3. E2E тест

    https://github.com/Agafocean/E2E_test

    E2E тест игры в пары.

    В Visual Studio Code нажмите Go Live.

    Выполните команду: npx cypress open.

    На открывшейся странице выберите E2E Testing.

    На следующей странице - Start E2E Testing.

    В списке слева выберите open-pairs.cy.js.
